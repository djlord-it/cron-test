# Data Demo

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

   Or run directly:
   ```bash
   node server.js
   ```

## Deploying to Railway

1. Push your code to a GitHub repository.

2. Go to [Railway](https://railway.app) and create a new project.

3. Select "Deploy from GitHub repo" and connect your repository.

4. Railway will automatically detect the configuration from `railway.json` and deploy your app.

5. Once deployed, Railway will provide you with a public URL for your application.
